1. How to create a login

To create a login, navigate to the login page and enter your username and password. After submitting, you'll be asked to confirm your account. If you don't have an account, you can create one by clicking the 'Create One' button.

Return to Top | Go to Login

2. Why create a login

Creating a login allows you to track your progress, save your goals, and access personalized features in the system. It's essential for managing and keeping track of your financial and habit-building progress.

Return to Top | Go to Login

3. How to log in

To log in, enter your registered username and password on the login page. If your credentials are correct, you'll be directed to your dashboard where you can view your plans and goals.

4. How to create a plan

To create a plan, navigate to the 'Create Plan' section from your dashboard. Provide a name and description for your plan, then submit it to start tracking.

Return to Top | Return to Dashboard | Create a Plan

5. Why create a plan

A plan helps you define your goals, break them into manageable steps, and track your progress. It is an essential tool for staying organized and motivated in both personal and financial goals.

Return to Top | Return to Dashboard | Create a Plan

6. What can a plan do

A plan allows you to organize tasks, set deadlines, and monitor progress towards completing your goals. You can add multiple goals to a plan and track them individually.

Return to Top | Return to Dashboard | Create a Plan

Section 3: Goals

  1. How to create a goal
  2. What can a goal do

7. How to create a goal

To create a goal, go to the 'Create Goal' section of your plan. Set a specific target, description, and timeline. Once submitted, the goal will be tracked within your selected plan.

Return to Top | Return to Dashboard | Create a Goal

8. What can a goal do

A goal allows you to track specific actions or achievements within a plan. Goals help break down large objectives into smaller, actionable steps, making it easier to measure progress.
